Research Group at Ontario Tech University

Current

  1. Data Driven Relational Constraint Database Systems - Masters Thesis - Ontario Tech University
    • A framework for optimization problems to be performed with iterative debugging that ensures data quality and safety when performing data-driven decision making from databases.
    • Accepted at the 2020 IEEE International Conference on Information Reuse and Integration.
    • Technologies:
      • Interactive Environment: IPython Notebook/Kernels
      • Languages: Clojure, Python, SQL
      • AI Engine: Google OR-Tools SAT Solver
      • Visualization: Matplotlib, Incanter
      • Data Structures: EDN, Google Protobuf
      • Deployment: Google Cloud, Docker, Linux

Past

  1. CdnPoli Twitter Analysis - Ontario Tech University
    • Performed analysis of the continuous #cdnpoli tweets to predict the standings in the Canadian 2019 election.
    • Technologies:
      • Interactive Environment: IPython Notebook, Bokeh
      • Databases: PostgreSQL
      • Languages: Python, SQL, JSON
      • ML/DL Frameworks: TensorFlow
      • Visualization: Bokeh, Matplotlib
      • Deployment: Linux, Docker
  2. Scalable Analysis of Open Data Graphs - Ontario Tech University
    • Performed analysis on the Canadian Open Data to improve insights into open data sources.
    • Published at the 2019 IEEE International Conference on Information Reuse and Integration.
    • Technologies:
      • Distributed Computing: Apache Spark (PySpark)
      • Databases: PostgreSQL, GCP Storage
      • Languages: Python, SQL, JSON
      • Visualization: Matplotlib
      • ML/DL Frameworks: scikit-learn, NLTK
      • Deployment: Google Cloud, Dataproc
